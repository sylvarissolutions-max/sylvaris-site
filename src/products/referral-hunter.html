What it does

Six operational functions, fully automated.

Discovery

Bounty discovery

Tracks public referral programs across tech companies. Ingests bounty details (company, role, amount, terms, expiration) into a structured database.

Matching

Candidate matching

Scores candidates against open bounties based on skills, experience, location, and fit signals. Surfaces the highest-probability matches first.

Pipeline

9-stage pipeline

Tracks candidates through: identified, contacted, responded, interested, submitted, interviewing, offer, placed, paid. Stage changes logged to activity.

Outreach

Outreach sequencing

Logs all communication (email, LinkedIn, Twitter, GitHub) with direction (sent/received) and outcome tracking. Planned: automated sequence triggers.

Placements

Placement and commission tracking

Records placement fee, probation period, payout date, invoice number, and payment status. Revenue dashboard with pipeline value and earned/pending split.

Activity

Full audit trail

Every bounty creation, candidate add, stage change, outreach log, and placement is recorded with timestamp and entity reference.

Current state

Platform

Standalone SaaS

Node.js + SQLite backend, Vite-built dashboard, JWT auth, rate limiting, deployed on Render.

Roadmap

Planned automation

Web scraping for new bounty discovery, automated LinkedIn outreach via browser extension, candidate import from GitHub contributor graphs.
