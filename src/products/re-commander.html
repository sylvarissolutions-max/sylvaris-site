What it does

Six operational functions, fully automated.

Ingest

Property ingest

Bulk import properties from MLS feeds, spreadsheets, or manual entry. Each property gets a structured profile with comps, zoning, and tax history.

Underwriting

Automated underwriting

ARV calculation, rehab cost estimation, and deal scoring based on configurable investment criteria. Flags deals that meet target returns.

Offers

Offer generation

Draft purchase offers with configurable terms, contingencies, and escalation clauses. Exportable as PDF for submission.

Leads

Lead scoring and outreach

Motivated seller leads scored by urgency signals. Automated outreach sequences via email and direct mail.

Pipeline

Deal pipeline

Visual deal tracker from lead to close. Stage gates, document management, and activity logging per deal.

Analytics

Portfolio analytics

Acquisition velocity, pipeline value, close rate, average hold time, ROI per deal. Exportable monthly reports.

Architecture

Built for isolation, secured by default.

Auth

Double-layer authentication

HTTP Basic Auth gate wraps the entire application. JWT sessions inside for granular access control. No token-only attacks possible.

Tenants

Multi-tenant isolation

Every database query enforces tenant_id. Complete data separation between investor accounts.

AI

AI-powered analysis

Natural language commander interface for asking questions about the portfolio. Comparable analysis and market narrative generation.
